# Drowsiness Detection System

ระบบตรวจจับความง่วงนอนโดยใช้เทคโนโลยีการมองเห็นด้วยคอมพิวเตอร์ (Computer Vision)

## 📋 ภาพรวมโครงการ

โปรเจกต์นี้พัฒนาระบบเพื่อตรวจจับเมื่อผู้ขับขี่รู้สึกง่วงนอนหรือเหนื่อยล้าจากการอ่านสภาพหน้า ตา และลักษณะการเคลื่อนไหว เพื่อเพิ่มความปลอดภัยในการขับขี่

## 🎯 วัตถุประสงค์

- ตรวจจับสัญญาณของความง่วงนอนแบบเรียลไทม์
- เพิ่มความปลอดภัยของผู้ขับขี่
- ใช้โมเดล Deep Learning สำหรับการวิเคราะห์เมตริกต่างๆ

## 🔧 เทคโนโลยีที่ใช้

- **Jupyter Notebook** (99.2%) - สำหรับการวิเคราะห์ข้อมูลและการทดลอง
- **Python** (0.8%) - สำหรับการพัฒนาและประมวลผล
- **Computer Vision Libraries**: OpenCV, TensorFlow/PyTorch
- **Deep Learning**: CNN models สำหรับการตรวจจับใบหน้าและตา

## 📊 การทำงาน

### ขั้นตอนการประมวลผล

1. **การจับภาพ (Capture)** - จับภาพจากกล้อง/วิดีโอ
2. **การตรวจจับใบหน้า (Face Detection)** - ใช้ Haarcascade หรือ YOLO
3. **การตรวจจับตา (Eye Detection)** - ตรวจจับตำแหน่งและสถานะตา
4. **การวิเคราะห์ (Analysis)** - วิเคราะห์ระยะเวลาปิดตา นำ้ตา และการกระพริบตา
5. **การตัดสินใจ (Decision)** - ตัดสินว่าผู้ใช้ง่วงนอนหรือไม่
6. **การแจ้งเตือน (Alert)** - ออกเสียงแจ้งเตือนหากตรวจจับความง่วงนอน

### เมตริกการตรวจจับ

- **Eye Closure Ratio**: สัดส่วนของเวลาที่ตาปิด
- **Blink Rate**: ความถี่ของการกระพริบตา
- **Head Position**: ตำแหน่งของศีรษะ
- **Facial Landmarks**: จุดสำคัญบนใบหน้า
